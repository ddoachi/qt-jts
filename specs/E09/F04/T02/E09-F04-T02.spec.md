## Executive Summary

Create the StrategyListWidget for displaying and navigating the strategy library. Features search/filter functionality, list view with strategy summaries, and selection signals for loading strategies in the editor.

## Execution Flow

```
1. Create StrategyListWidget(QWidget)
   → Contains search box and list view
2. Add search functionality
   → QLineEdit with placeholder
   → Filter as user types
3. Display strategy summaries
   → Show name, entry rule preview
   → Show version and active status
4. Emit selection signals
   → strategy_selected(strategy_id: int)
   → Return: SUCCESS with list widget ready
```

## User Stories

### Primary User Story
**As a** trader
**I want to** browse my strategy library
**So that** I can find and select strategies to edit

## Acceptance Scenarios

### Scenario 1: Display Strategies
**Given** 5 strategies in repository
**When** widget is loaded
**Then** all 5 strategies shown in list

### Scenario 2: Search Strategies
**Given** strategy "RSI Momentum" in list
**When** user types "RSI" in search
**Then** only matching strategies shown

### Scenario 3: Select Strategy
**Given** strategy list with items
**When** user clicks a strategy
**Then** strategy_selected signal emitted
